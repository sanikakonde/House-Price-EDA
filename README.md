# 🏡 House Sales Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the King County House Sales dataset using Python.

The main goal of this analysis is to understand the dataset and identify the features that have the biggest impact on house prices. These insights can be used later to build a machine learning model for predicting house prices.

---

## Objective

- Clean and understand the dataset.
- Explore the distribution of different features.
- Find relationships between house price and other features.
- Identify the most important features that affect house prices.
- Prepare the data for future house price prediction models.

---

## Tools & Libraries

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn

**King County House Sales Dataset**
